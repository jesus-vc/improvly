## Overview

TL;DR: Cross-platform mobile app (Android & iOS) that uses LLMs, LMMs, and Retrieval-Augmented Generation (RAG) to turn everyday observations into funny, creative comedy material.

Become funnier in real life, more charming in conversation, and more confident in the moments that matter most.

## 🚧 Current Development Phase (March 2026)

The project is currently in the **architecture and foundation stage**.

Current focus areas:
- Documenting system design through Architectural Decision Records (ADRs)
- Designing the core database schema
- Defining the RAG pipeline and AI service interfaces
